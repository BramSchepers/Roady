# Roady

Statische website voor Roady – rijbewijs theorie.

## Opzet

- **Statische site** in `static_website/` (HTML, CSS, JS). Bevat o.a. `index.html`, `nl/`, `fr/`.
- **Deploy naar Firebase Hosting:** `.\deploy.ps1` (PowerShell). Deployt alleen de statische site naar https://goroady-22332.web.app.

## Referentie laadscherm

In `static_website/reference_splash/` staat een kopie van het oude laadscherm (Lottie-autootje) uit de voormalige Flutter-app, om later te hergebruiken bij een nieuwe /auth.
