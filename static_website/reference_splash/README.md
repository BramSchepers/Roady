# Referentie: laadscherm (splash) van de oude Flutter-app

Dit was het laadscherm met het Lottie-autootje. Je kunt dit later hergebruiken bij een nieuwe /auth (bijv. als HTML + Lottie-player, of in een andere stack).

- **splash_screen.dart** – Flutter-code: groene achtergrond #e8f0e9, Lottie-auto, 2,5 s delay, daarna navigatie naar /auth.
- **car.lottie** – De Lottie-animatie van het autootje.

Lottie op het web: https://lottiefiles.com of gebruik `lottie-web` / `@lottiefiles/dotlottie-web`.
